SCL瀑布流整体比SC下降陡一些

#### SCL下的最优p结构

| SNR区间            | SC最优p*             | SCL最优p*            | 变化                           |
| ------------------ | -------------------- | -------------------- | ------------------------------ |
| ≤4 dB             | 0.45–0.50           | 0.45–0.50           | 不变                           |
| **5–13 dB** | **0.10–0.16** | **0.10–0.16** | **不变**，仍是整形增益区 |
| ≥14 dB            | 0.45–0.50           | 0.50                 | SCL加速了p=0.5的瀑布           |

三段式结构在SCL下得以保留，整形增益区仍为5–13 dB，为下一步多载波优化奠定基础。

#### SCL 对系统的整体影响

| 指标 | SC | SCL | 改善 |
|------|-----|------|------|
| BER=0 所需 SNR (p=0.5) | 24 dB | 21 dB | **3 dB** |
| BER=0 所需 SNR (p=0.3) | ~23 dB | 22 dB | **1 dB** |
| 瀑布中心右移量 | — | — | 约 **2 dB 左移** |
| 计算开销 | 1× | 1.4× | 可接受 |

| 成果                      | 说明                                                               |
| ------------------------- | ------------------------------------------------------------------ |
| **三段式最优p**     | 低SNR(码率)→中SNR(整形)→高SNR(码率)，为自适应切换提供理论基础    |
| **SCL -3 dB增益**   | 列表译码显著改善BER，瀑布左移2 dB，为论文提供对比亮点              |
