## 一、项目总览

### 1.1 研究主题

在 6G 无电池物联网场景下，从编码与波形层面实现**数据与能量的物理层内生融合**：
- 以极化码概率整形调控符号分布；
- 以自适应多载波波形提升整流效率；
- 最终在 USRP 平台完成端到端验证。

### 1.2 三条主线

| 主线 | 核心内容 | 交付形式 |
|------|---------|---------|
| **主线一** | 16QAM 单载波概率整形极化码设计 | 仿真框架 + 权衡曲线 + 论文 |
| **主线二** | 多载波 OFDM 子载波级动态数能分配 | 优化算法 + 仿真验证 + 论文 |
| **主线三** | 自适应内生接收机 + USRP 端到端验证 | 硬件原型 + 实测数据 + 毕业论文 |

### 1.3 已完成工作摘要（截至 2026.03）

- [x] 16QAM Gray 映射 LLR 推导与验证（LSE vs max-log）
- [x] 单载波概率整形极化码框架搭建（四路并行编码 → 串行调制 → AWGN → 四路独立译码）
- [x] 核心指标体系：Goodput $G(p)=R(p)(1-\text{BER}(p))$、能量 $E(p)=18-16p$、代价 $\text{cost}(p)=-\Delta G/\Delta E$
- [x] 三种优化形式梳理：加权和 / 约束型 / 代价函数
- [x] $p$ 与星座均方能量的解析关系与归一化
- [x] 互信息分析（无信道 vs 有信道、同位置 MI 局限性）
- [x] GA 虚拟信道可靠性估计及其与整形位的冲突分析

## 三、阶段规划（中长期）

### 阶段 A：单载波系统收官（2026.03 – 2026.05）

> **目标**：产出完整的 16QAM 概率整形极化码单载波系统性能分析，形成可投稿论文素材。

#### A1. 系统性 BER-能量 Pareto 扫参（3 月上旬）

- 在 $p \in \{0.50,0.45,0.40,0.35,0.30,0.25,0.21,0.16,0.13,0.10\}$、$\text{SNR}\in\{-5,-2,0,2,4,6,8,10,12,15,18,20\}$ dB 网格统一跑仿真；
- 记录 BER、Goodput、$E(p)$、互信息 $I(B_k;L_k)$；
- 输出 Pareto 前沿图（Goodput vs $\tilde E$）。

**验收条件**：
- 生成 `results/YYYYMMDD_pareto_sweep/`，包含 `params.mat`、`results.mat`、`figures/*.pdf`；
- Pareto 曲线可清晰标注最优 $p^*$ 区间。

#### A2. SCL 译码器非均匀先验适配（3 月中旬）

- 审查 SCL 路径度量是否引入非均匀先验 $P(u_i)$；
- 若未适配，在 PM 更新中加入 $\log P(u_i=b)$ 偏置；
- 对比适配前后 BER 与 Goodput（重点关注 $p\le 0.21$）。

#### A3. 整流器模型引入（3–4 月）

- 建立二阶/三阶多项式或分段线性整流模型，替代“符号均方”能量指标；
- 计算 $\eta_{\text{rect}}(p)$ 并并入 Pareto 分析。

#### A4. 论文一初稿（4–5 月）

- 形成“16QAM 概率整形极化码数能同传性能分析”论文初稿；
- 目标投稿：ICC/GLOBECOM 或 IEEE CL/WCL。

---

### 阶段 B：多载波系统构建（2026.05 – 2026.09）

阶段 B 当前按“先链路、再画像、再策略”的小阶段推进，避免把 OFDM 实现、Rayleigh 画像和 `p_k` 优化混在同一次实验中判断。

- **B1：OFDM AWGN baseline 与可视化诊断**  
  建立统一 `p` 的 OFDM full-chain baseline，固定 AWGN、SC decoder、CP 修正 Goodput；输出 BER、Goodput、Energy-Goodput 曲线，并补充 PSD、频域资源网格和星座图诊断。B1 只验证 OFDM 链路闭环，不进入 Rayleigh 和 `p_k` 优化。

- **B2：Rayleigh 子载波可靠性画像**  
  在 OFDM 基础上引入 Rayleigh 多径信道，输出每个子载波的 `|H_k|^2`、等效 $\gamma_k$、MI proxy 和 high/mid/low 可靠性分组；该阶段只形成可靠性画像和分组依据，不直接判断哪种整形策略最优。

- **B3：统一 `p` baseline 与三类子载波策略对比**  
  基于 B2 分组，至少比较四类口径：统一 `p` baseline、好信道偏信息、好信道偏能量整形、差子载波纯传能。比较指标包括 BER、Goodput、接收侧能量 proxy、信息子载波占比和分组阈值。

- **B4：低复杂度优化算法**  
  在 B3 统一指标口径稳定后，再研究查表、贪心、分组优化或近似凸松弛等低复杂度方案，重点控制复杂度和信令开销。

- **B5：论文二材料整理与初稿**  
  汇总 Rayleigh OFDM 可靠性画像、三类策略对比、低复杂度算法和边界条件，形成多载波概率整形极化码论文二素材，目标 TVT/TCOM/TWC。

## 五、风险识别与应对

| 风险 | 等级 | 应对策略 |
|------|------|---------|
| 强整形下 SCL 性能崩塌 | 高 | 引入 CA-SCL/先验修正，限制 $p$ 下界 |
| 整流器模型偏差过大 | 中 | 文献模型先行，硬件阶段实测校准 |
| OFDM 逐子载波编码复杂度过高 | 中 | 子载波分组 + 查表法 |
| USRP 实测与仿真差距 > 3 dB | 高 | 分阶段上板 + 硬件损伤标定 + 算法容限 |
| 论文投稿周期影响进度 | 低 | 会议与期刊并行投 |
| 硬件采购延迟 | 中 | 提前采购，保留纯仿真最低可行交付 |
